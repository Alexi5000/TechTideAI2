# Tool: market-intel

**Description**: Captures competitive and market research insights with citations.

**Inputs (JSON)**
```json
{
  "query": "string",
  "scope": "competitors | industry | macro",
  "recencyDays": 90
}
```

**Outputs (JSON)**
```json
{
  "summary": "string",
  "citations": ["string"]
}
```
