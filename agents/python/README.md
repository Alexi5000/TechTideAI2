# Python Agents Toolkit

This package hosts LangGraph/LangChain utilities for TechTideAI.

## Setup
```bash
python -m venv .venv
. .venv/Scripts/activate  # Windows PowerShell: .venv\Scripts\Activate.ps1
pip install -e .[dev]
```

## Run the sample graph
```bash
python -m techtide_agents.graph
```

## Tests
```bash
pytest
```
